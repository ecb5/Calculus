Problem 21.7.0.27.

Actually, Wolfram Alpha provides the following output
\begin{equation*} \frac{1} {\sqrt{2\pi}}\int^{8/3}_{z=-\infty} {e^{-\frac{1}{2}z^2}}\dx{z} =\frac{1}{2}\left[\erf\left(\frac{4\sqrt{2}}{3}\right)+1\right] \approx .99617 \end{equation*}
where
\begin{equation*} \erf \left(a\right)=\frac{2}{\sqrt{\pi}}\int^a_{t=0}{e^{-t^2}\dx{t}} \end{equation*}
The name erf is short for error function and reminds us that Gauss determined this function to compute the probability of errors in measurements.
