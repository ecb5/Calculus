Cornerstone Problem 19.2.0.1.

In part (c) of Problem 18.2.0.8 you were faced with the problem of computing
\begin{equation} \int\frac{1}{u(u+1)} \dx{u}\tag{19.34} \end{equation}
which is relatively simple to do if you know that
\begin{equation} \frac{1}{u(u+1)}=\frac1u-\frac{1}{u+1}\tag{19.35} \end{equation}
This is why we had you confirm formula (19.35) in part (d). But without this decomposition computing the integral would have been daunting at best. The process by which we compute the decomposition \(\frac{1}{u(u+1)}=\frac1u-\frac{1}{u+1}\) is known as the Partial Fraction Decomposition (PFD). The PFD is useful in a variety of situations beyond the computation of integrals, but it is a purely algebraic operation so we will not be examining its deeper implications. For us, right now, it is only useful as a way to decompose integrals like formula (19.34).

(a)

We will build up the computational scheme slowly, starting with simple decompositions. We’ll begin by making the simplest possible guess for the decomposition so we assume that the decomposition has the form:
\begin{equation} \frac{1}{u(u+1)} = \frac{A}{u} + \frac{B}{u+1}.\tag{19.36} \end{equation}
