Example A.13.2.8.

Suppose we need to compute
\begin{equation*} \int{\frac{25}{{\left(x-2\right)}^2(x^2+1)}}\ \dx{x} \end{equation*}
Applying our partial fractions decomposition, we get
\begin{equation*} \frac{25}{{\left(x-2\right)}^2\left(x^2+1\right)}=\frac{Ax+B}{{\left(x-2\right)}^2}+\frac{Cx+D}{x^2+1} \end{equation*}
\begin{equation*} =\frac{\left(Ax+B\right)\left(x^2+1\right)+\left(Cx+D\right){\left(x-2\right)}^2}{{\left(x-2\right)}^2\left(x^2+1\right)} \end{equation*}
\begin{equation*} =\frac{\left(A+C\right)x^3+\left(B-4C-D\right)x^2+\left(A+4C+4D\right)x+\left(B-4D\right)}{{\left(x-2\right)}^2\left(x^2+1\right)} \end{equation*}
🔗
This leads to the equations
\begin{align*} A+C\amp{}=0\\ B-4C-D\amp{}=0\\ A+4C+4D\amp{}=0\\ B-4D\amp{}=25 \end{align*}
Solving this (any way you can), we get \(A=-4\text{,}\) \(B=13\text{,}\) \(C=4\text{,}\) and \(D=-3\text{.}\)
🔗
Getting back to the integral, we have
\begin{equation*} \int{\frac{25}{{\left(x-2\right)}^2\left(x^2+1\right)}}\ \dx{x}=\int{\frac{-4x+13}{{\left(x-2\right)}^2}}\dx{x}+\int{\frac{4x-3}{x^2+1}\dx{x}} \end{equation*}
\begin{equation*} =\int{\frac{-4x+13}{{\left(x-2\right)}^2}}\dx{x}+4\int{\frac{x}{x^2+1}\dx{x}}-3\int{\frac{1}{x^2+1}\dx{x}} \end{equation*}
\begin{equation*} =\int{\frac{-4x+13}{{\left(x-2\right)}^2}}\dx{x}+2{\mathrm{ln} \left(x^2+1\right)\ }-3{\mathrm{arctan} x\ }+c \end{equation*}
Notice that the last two integrals were pretty straightforward. This first integral is more problematic. There is a little algebraic trick which will allow us to break up that integral even further.
\begin{equation*} \int{\frac{-4x+13}{{\left(x-2\right)}^2}}\dx{x}=\int{\frac{-4\left(\left(x-2\right)+2\right)+13}{{\left(x-2\right)}^2}}\dx{x} \end{equation*}
\begin{equation*} =\int{\frac{-4\left(x-2\right)-8+13}{{\left(x-2\right)}^2}}\dx{x} \end{equation*}
\begin{equation*} =\int{\frac{-4\left(x-2\right)+5}{{\left(x-2\right)}^2}}\dx{x} \end{equation*}
\begin{equation*} =-4\int{\frac{1}{x-2}\ \dx{x}}+5\int{\frac{1}{{\left(x-2\right)}^2}}\dx{x} \end{equation*}
\begin{equation*} =-4{\mathrm{ln} \left(x-2\right)\ }-5{\left(x-2\right)}^{-1} \end{equation*}
🔗
Putting this all together, we finally have
\begin{equation*} \int{\frac{25}{{\left(x-2\right)}^2\left(x^2+1\right)}}\ \dx{x}=-4{\mathrm{ln} \left(x-2\right)\ }-5{\left(x-2\right)}^{-1}+2{\mathrm{ln} \left(x^2+1\right)\ }-3{\mathrm{arctan} x\ }+c \end{equation*}
The trick that we employed above is something you might want to remember. It allows us to take our partial fractions decomposition a bit further. For example, if we start with
\begin{equation*} \int{\frac{x^3}{{\left(2x^2+1\right)}^2}}\ \dx{x} \end{equation*}
we could rewrite this as
\begin{align*} \int{\frac{x^3}{{\left(2x^2+1\right)}^2} \dx{x}}&=\int{\frac{\frac{1}{2}x\left(2x^2\right)}{{\left(2x^2+1\right)}^2} \dx{x}}\\ &=\int{\frac{\frac{1}{2}x\left(2x^2+1-1\right)}{{\left(2x^2+1\right)}^2} \dx{x}}\\ &=\frac{1}{2}\int{\frac{x}{2x^2+1}\ \dx{x}}-\frac{1}{2}\int{\frac{x}{{\left(2x^2+1\right)}^2}}\ \dx{x} \end{align*}
