Problem 8.3.0.14. The Exponential Sum Property.

Since we’ve encouraged you to think of the expression \(e^t\) as the funny number \(e\) raised to the power \(t\text{,}\) it seems obvious that the rules of exponents apply so that \(e^{t+a} = e^t\cdot e^a\text{.}\)
🔗
But nothing we’ve said so far actually makes this a true statement. Definition 8.2.0.2 names the function and states one of its properties (that it is its own derivative), but that is all. This problem shows that we can use the definition to conclude that \(e^{t_a} = e^t\cdot e^a\) must also be true.
🔗

(a)

Show that \(y=e^{t+a}\) satisfies \(\dfdx{y}{t}=y\) for any constant \(a\text{.}\)
🔗
Hint.
Recall Definition 8.2.0.2.
🔗
🔗
🔗

(b)

Show that \(y=Ae^t\) also satisfies \(\dfdx{y}{t}=y\) for any constant \(A\text{.}\)
🔗
🔗

(c)

Part 8.3.0.14.a and part 8.3.0.14.b imply that \(e^{t+a}=Ae^t\text{.}\) Use this knowledge to to show that
\begin{equation*} e^{t+a}=e^t\cdot e^a. \end{equation*}
