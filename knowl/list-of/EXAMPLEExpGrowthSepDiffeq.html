Example 18.2.0.4.

We spent a great deal of effort in Section 8.6 analyzing the equation
\begin{equation} \dfdx{y}{t} = Ky\tag{18.15} \end{equation}
for various values of \(K\text{.}\) But in that section our efforts were based on guessing the form of the solution. But equation (18.15) is separable so we no longer need to guess.
🔗
To keep things simple for now we’ll take \(K=1\text{.}\) That is, we want to solve
\begin{equation*} \dfdx{y}{t}=y\text{.} \end{equation*}
🔗
First we separate the variables
\begin{equation*} \frac{\dx{y}}{y}=\dx{t}\text{,} \end{equation*}
then we integrate both sides, giving
\begin{equation*} \ln\abs{y} = t + C\text{.} \end{equation*}
Solving for \(\abs{y}\) next we get
\begin{equation*} \abs{y} = Ke^t \end{equation*}
where \(K=e^C\text{.}\) Finally, since this is a model of a population and populations are never negative we see that the absolute value bars are superfluous here so
\begin{equation*} y(t) = Ke^t \end{equation*}
where \(K=e^C\text{,}\) just as we saw in Section 8.6.
