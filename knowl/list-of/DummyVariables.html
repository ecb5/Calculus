Dummy Variables.

Did you notice that in all of our computations in Example 20.2.0.1 we (apparently) used two variables? In part (c) for instance we used \(t\) inside the integral but we used \(x\) as the variable for the accumulation function. Do we really have two variables in play here as it appears we do or is something else happening? Give this some thought before you read on.
🔗
In the caption of Figure 20.2.0.2 we were careful to point out that the horizontal axis represents \(t\text{,}\) not \(x\text{.}\) The variable \(x\) represents the position on the \(t-\)axis where we stop accumulating area. Thus \(F\) is a function of \(x\text{,}\) not \(t\text{.}\)
🔗
To put it a little differently the role of \(t\) is to define the function \(f(t)\) and we use \(f(t)\) to define \(F(x)\) as an integral. The role of \(x\) is to locate the position on the \(t\)–axis where we stop summing the areas of the differential rectangles. But since \(F(x)\) depends on the value of \(x\) it is the variable for \(F\text{.}\) This can be confusing. Be careful.
🔗
In fact, in a real sense the variable \(t\) isn’t really present. That it appears to be is an artifact of our having defined \(F(x)\) via an integral. When we do this we must have another function to integrate. This forces us to use a different variable name to keep things orderly.
🔗
To see that \(t\) isn’t really present consider the function
\begin{equation*} F(x) = \int_{t=1}^xt\dx{t}\text{.} \end{equation*}
Evaluating, we have
\begin{equation*} F(x) = \eval{\frac{t^2}{2}}{t}{1}^x = \frac{x^2}{2}-\frac{1}{2} \end{equation*}
or
\begin{equation*} F(x) = \frac{x^2}{2}-\frac{1}{2} \text{.} \end{equation*}
Notice that there is no \(t\) present in the last formula. The function \(F\) depends on \(x\) not \(t\text{.}\) The integration variable (\(t\) in this case) is called a dummy variable. It is “dumb” in the old–fashioned sense that it is mute.
