The Antiderivative is Not Unique.

As we observed earlier the antiderivative of a function is not unique. This fact will cause us a multitude of headaches if we don’t address it now.
🔗
Consider the differential equation
\begin{equation*} \dx{y}=2x\dx{x}\text{.} \end{equation*}
Clearly \(y=x^2\) is a solution because \(\dx{y}=\dx(x^2) = 2x\dx{x}\) as required. But by the same reasoning \(y=x^2+1\) is also a solution. As is \(y=x^2+20\text{.}\) In fact, if \(C\) is any constant whatsoever then \(y=x^2+C\) will be a solution of \(\dx{y}=2x\dx{x}\) because when we differentiate \(x^2+C\) we get
\begin{equation*} \dx (x^2 +C) = \dx (x^2) + \dx (C) = 2x\dx{x}\text{.} \end{equation*}
Thus we see that
\begin{equation*} \int 2x\dx{x} =x^2+C \end{equation*}
is true regardless of the value of the constant \(C\text{.}\)
🔗
We will use the expression \(\int f(x)\dx{x}\) to represent all possible antiderivatives of \(f(x)\text{.}\) It is called the most general antiderivative. We formalize all of this in the following definition.
🔗

Aside: Mathematics Terminology.

Because it does not have a definite value the most general antiderivative is also sometimes called the indefinite integral of \(f(x) \text{.}\) But in this text we will avoid that usage.
🔗

Definition 18.1.0.2. The Most General Antiderivative.

Given a function \(f(x) \) the expression
\begin{equation*} \int f(x) \dx{x} \end{equation*}
represents the most general antiderivative of \(f(x) \text{.}\) It is also sometimes called the indefinite integral of \(f(x) \text{,}\) though we will not be doing so in this text.
🔗
The function \(f(x)\) appearing between \(\int{}\) and \(\dx{x}\) is called the integrand of the integral.
🔗
🔗
Because \(C\) can be any constant the expression \(\int f(x)\dx{x}\) represents, strictly speaking, a multifunction We first encountered multifunctions in Chapter 6 and they were rather difficult to work with because of the complex relationships between the domain and range of a trig function and its inverse. That isn’t a problem here. The distinct branches (functions) of the multifunction \(\int{}f(x)\dx{x}\) are distinguished by selecting a different value of the arbitrary constant.
🔗
As a practical matter what this means for you is that if you are asked to evaluate an integral, say \(\int{}x^2\dx{x}\text{,}\) you may be tempted to write down \(\frac{x^3}{3}\text{.}\) But this is wrong because it is the particular branch of the multifunction \(\frac{x^3}{3}\) where the constant is zero. Since we need all of the branches the solution of this problem is
\begin{gather*} \int{}x^2\dx{x}=\frac{x^3}{3} +C \end{gather*}
where \(C\) is understood to be an undetermined constant.
🔗
For now we simply want to develop skill at evaluating antiderivatives so we will not fuss too much about multifunctions. But be aware that if you are asked to evaluate an antiderivative like \(\int{}f(x)\dx{x}\) you are being asked to find the most general antiderivative of \(f(x) \text{.}\) This means that you need to tack on that “\(+ C\)” at the end. Leaving it out is incorrect.
🔗
Integration gives us a more systematic way of solving differential equations (and, eventually, Initial Value Problems) than we have now. For instance suppose we need to solve the differential equation
\begin{equation*} \dfdx{y}{x} = x-e^x\text{.} \end{equation*}
We first multiply through by \(\dx{x}\) and then integrate both sides, giving
\begin{equation} \int \dx{y}=\int y(x) =x-e^x\dx{x}.\tag{18.7} \end{equation}
🔗
Next we need to find an antiderivative of \(f(x)=x-e^x\) which we can clearly build from the antiderivatives of \(x\) and \(e^x\text{.}\) An antiderivative of \(x\) is \(\frac{x^2}{2}\) and \(e^x\) is its own antiderivative. So an antiderivative of \(x-e^x\) is \(\frac{x^2}{2}-e^x\) and thus
\begin{equation*} \int x-e^x\dx{x} = \frac{x^2}{2} - e^x +C \end{equation*}
(where \(C\) is an unspecified constant) because \(\frac{x^2}{2} - e^x+C\) is the most general antiderivative of \(f(x)=x-e^x\text{.}\)
🔗
