Example 17.2.3.1.

Suppose \(f(x) = -2-\frac{1}{x}.\) Can you guess the value of \(\limit{x}{\infty}{f(x)}\text{?}\) Let \(\eps\gt 0\) be given. Then when \(B\) has any positive value, if \(x\gt B\) we have
\begin{equation*} f(x) = -2-\frac1x \lt 0 \lt \eps, \end{equation*}
and by Definition 17.2.2.1 we conclude that \(\tlimit{x}{\infty}{f(x)} =0\) since all of the requirements of our definition have been met (except, of course, \(f(x)\gt0\)). Of course, this is nonsense. As we have noted as \(x\) increases without bound \(\frac1x\) gets arbitrarily close to \(0\text{.}\) So clearly \(\tlimit{x}{\infty}{f(x)}=-2\text{.}\) That we are able to “prove” that \(f(x)\) goes to zero simply means that Definition 17.2.2.1 doesn’t capture everything we need. We need a more encompassing definition of a limit.
