Example 18.2.0.1. Separating Variables in Algebra.

Separating variables is not a new idea. You’ve been using it all of your life to solve algebraic equations. For example, suppose we want to find all solution of the algebraic equation
\begin{equation} y^2-x^2y - 6y=0\tag{18.10} \end{equation}
for \(y\text{.}\) Some of these solutions are easy to find. For example the set of all points with coordinates \((x,y)=(x,0)\) are solutions. But these are not all of the solutions. To find the others we assume that \(y\neq 0\text{,}\) and divide by \(y\) obtaining
\begin{equation*} y-x^2-6=0\text{.} \end{equation*}
Next we separate the variables by rearranging the terms algebraically so that all of the terms involving \(y\) are on on one side, and all of the terms involving \(x\) (or involving neither \(x\) nor \(y\)) on the other
\begin{equation*} y=x^2+6\text{.} \end{equation*}
So the set of all points with coordinates \((x,x^2+6)\) are the remaining solutions of equation (18.10).
