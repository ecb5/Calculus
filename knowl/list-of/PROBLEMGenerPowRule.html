Problem 8.12.4.8. The General Power Rule.

We are now, finally, in a position to extend the Power Rule for Positive, Rational Exponents 4.3.5.20 to irrational exponents. In fact we’ve already done it.
🔗

(a)

Explain how the results of Problem 8.12.4.7 imply that if
\begin{equation*} y=x^\alpha{} \end{equation*}
and \(x\gt 0\) then
\begin{equation} \dx{y}=\alpha x^{\alpha -1}\dx{x}\tag{8.31} \end{equation}
even when \(\alpha{}\) is irrational.
🔗
🔗

(b)

Why did we impose the restriction \(x\gt 0\text{?}\)
🔗
Hint.
What happens if \(x\lt 0\) and \(\alpha = 1/2\text{?}\)
🔗
🔗
🔗
Since we did not constraint \(\alpha\) except to say that it is a real number the different versions of the Power Rule that we saw in Section 4.3 are all special cases of equation (8.31).
