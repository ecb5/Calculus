Problem 21.6.0.19.

In Problem 21.3.0.22 you were asked to find the volume of the solid generated by revolving one arch of a cycloid given by
\begin{equation*} x=a(\theta -\sin( \theta ), \text{ } y=a(1-\cos(\theta ) \end{equation*}
around the \(x\)-axis.
🔗
The answer to that problem was \(5\pi^2a^3\text{.}\) We still haven’t proved Pappus’ Theorem yet, but compute \(\bar{y} \) for this planar region and see if Pappus’ Theorem applied to this will yield the same answer.
