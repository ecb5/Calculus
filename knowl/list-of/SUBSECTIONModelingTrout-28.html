Problem 11.1.1.5.

In this problem we analyze the concavity of the graph of \(P(t)\text{.}\) In particular we’d like to find all values of \(P\) where PITPs occur.
🔗

(a)

Show that
\begin{equation*} \dx\left(\dfdx{P}{t}\right)=0.1\left(1-\frac{P}{50}\right)\dx{P} \end{equation*}
and therefore
\begin{equation*} \dfdxn{P}{t}{2}=0.1\left(1-\frac{P}{50}\right)\dfdx{P}{t}. \end{equation*}
🔗
🔗

(b)

Find all values of \(P\) where a PITP occurs.
🔗
🔗

(c)

Create a table similar to Table 11.1.1.3 to determine the (vertical) intervals on which \(P\) is concave up or concave down.
