Example 19.2.3.2.

Next consider the rational function \(\frac{1}{(x-1)^2(x+2)^2}\text{.}\) Its PFD will have two terms, one with denominator \((x-1)^2\) and the other with denominator \((x+2)^2\text{.}\) As before we’ll need to assume that these numerators are linear (have degree one less that the degree of their associated denominators; one in this instance). That is, the only requirement we have on these numerators is that they be linear polynomials. The obvious forms to use are \(\alpha{}x+\beta{}\) and \(\Gamma{}x+\Delta{}\) (where \(\alpha{}\text{,}\) \(\beta{}\text{,}\) \(\Gamma{}\text{,}\) and \(\Delta{}\) are the constants to be found) but it will turn our that those aren’t very helpful. If instead we write our numerators in the forms \(A(x-1)+B\) and \(C(x+2)+D\) things will simplify nicely.
🔗
So we seek constants \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) such that
\begin{gather*} \frac{1}{(x-1)^2(x+2)^2}= \frac{A(x-1)+B}{(x-1)^2}+\frac{C(x+2)+D}{(x+2)^2}. \end{gather*}
or
\begin{gather*} \frac{1}{(x-1)^2(x+2)^2}= \frac{A}{x-1}+\frac{B}{(x-1)^2}+\frac{C}{x+2}+\frac{D}{(x+2)^2}. \end{gather*}
🔗
By Heaviside’s Method we can see that
\begin{align} B\amp{}=\eval{ \frac{1}{(\text{covered})(x+2)^2}}{x}{1} = \frac{1}{9}\tag{19.51} \end{align}
and
\begin{align} D\amp{}=\eval{\frac{1}{(x-1)^2(\text{covered})}}{x}{-2}=\frac{1}{9}.\tag{19.52} \end{align}
And so we have
\begin{gather*} \frac{1}{(x-1)^2(x+2)^2}= \frac{A}{x-1}+\textcolor{red}{\frac{\frac{1}{9}}{(x-1)^2}}+\frac{C}{x+2}+\textcolor{red}{\frac{\frac{1}{9}}{(x+2)^2}}{} \end{gather*}
or
\begin{equation} \frac{1}{(x-1)^2(x+2)^2}-\textcolor{red}{\frac{\frac{1}{9}}{(x-1)^2}}{}-\textcolor{red}{\frac{\frac{1}{9}}{(x+2)^2}}{}= \frac{A}{x-1}+\frac{C}{x+2}\tag{19.53} \end{equation}
🔗
At this point the situation is similar to that of equation (19.49), except that this time we have two constants to find. If we proceed as we did there we see that
\begin{align*} \frac{1}{(x-1)(x+2)}-\frac{\frac{1}{9}(x+2)}{(x-1)}-\frac{\frac{1}{9}(x-1)}{(x+2)}\amp{}= A(x+2)+C(x-1)\\ \amp{}=(A+C)x+(2A-C). \end{align*}
The obvious next step is to multiply out the left side of the equation — we are guaranteed to get a linear polynomial — and compare the coefficients. But that is the procedure we rejected in favor of Heaviside’s Method back in Subsection 19.2.1 because it involves too much computation. We’d really like to find a simpler method.
🔗
Take another look at equation (19.53), especially the right side. Writing down that expression is exactly how we would begin the decomposition of a rational function of the form \(\frac{P(x)}{(x-1)(x+2)}\) (where \(P(x)\) is a polynomial we will determine shortly). Thus we have
\begin{equation} \frac{P(x)}{(x-1)(x+2)}=\frac{A}{x-1}+\frac{C}{x+2}\tag{19.54} \end{equation}
or equivalently,
\begin{equation} \frac{1}{(x-1)^2(x+2)^2}-\frac{\frac{1}{9}}{(x-1)^2}-\frac{\frac{1}{9}}{(x+2)^2}=\frac{P(x)}{(x-1)(x+2)}.\tag{19.55} \end{equation}
If we can find \(P(x)\) we can use Heaviside’s Method on equation (19.54) to find \(A\) and \(C\text{.}\) This will very helpful if we find \(P(x)\) very efficiently. And we can, although it probably isn’t clear how yet.
🔗
Proceeding, we clear the fractions giving
\begin{gather*} \underbrace{\frac{1}{(x-1)(x+2)}}_{=R_1}-\underbrace{\frac{\frac{1}{9}(x+2)}{x-1}}_{=R_2}-\underbrace{\frac{\frac{1}{9}(x-1)}{x+2}}_{=R_3}=P(x). \end{gather*}
This is similar to equation (19.50) except that this time we are guaranteed that the sum of the terms on the left will reduce down to some polynomial, not some constant
 3 
It’s actually exactly the same thing. A constant is a polynomial of degree zero.
. Do you see how to find the polynomial without doing all of that algebra? Think about it for a bit before going on.
🔗
We will consider each of the expressions \(R_1\text{,}\) \(R_2\text{,}\) and \(R_3\) separately.
\(R_1=\frac{1}{(x-1)(x+2)}\)
🔗
This expression is in what we will call “lowest terms” for purposes of this discussion. What we mean is that there is no algebraic operation we can perform on this expression — without changing its value — that will yield a polynomial. Thus we can ignore it.
🔗
\(R_2=\frac{\frac{1}{9}(x+2)}{x-1}\)
🔗
Next consider \(R_2=\frac{\frac{1}{9}(x+2)}{x-1}\text{.}\) If we perform the indicated division we get
\begin{align*} \amp{}\frac{\frac{1}{9}(x+2)}{x-1}\\ \amp{}\ \ \ \ =\frac{1}{9}+\text{some rational function in "lowest terms."} \end{align*}
We have not bothered to write down the rational function that appears from the division because, since it is in “lowest terms” we can ignore it, just like \(R_1\)
🔗
\(R_3=\frac{\frac{1}{9}(x-1)}{x+2}\)
🔗
Just like \(R_2\) this division will result in
\begin{align*} \amp{}\frac{\frac{1}{9}(x-1)}{x+2}\\ \amp{}\ \ \ \ =\frac{1}{9}+\text{some rational function in "lowest terms."} \end{align*}
where the function in “lowest terms” is not necessarily the same one we had before, but again since it will not yield a polynomial we can ignore it.
🔗
🔗
After ignoring the irrelevant terms identified above we have
\begin{align*} P(x) \amp{}= {R_1}\RedApproach{0}- {R_2}\RedApproach{1/9}-{R_3}\RedApproach{1/9} = -\frac{2}{9}, \end{align*}
which is a polynomial of degree \(0\text{.}\)
🔗
Combining this with equation (19.54) we have
\begin{gather*} \frac{-\frac{2}{9}}{(x-1)(x+2)}=\frac{A}{x-1}+\frac{C}{x+2} \end{gather*}
and we use Heaviside’s Method to determine that
\begin{align} A\amp{}=\eval{\frac{-\frac{2}{9}}{(\text{covered})(x+2)}}{x}{1}=-\frac{2}{27}\tag{19.56} \end{align}
and
\begin{align} C\amp{}=\eval{\frac{-\frac{2}{9}}{(x-1)(\text{covered})}}{x}{-2}=\frac{2}{27}.\tag{19.57} \end{align}
Combining equations (19.51), (19.52), (19.56), and (19.57) we have
\begin{equation} \frac{1}{(x-1)^2(x+2)^2}= \frac{-\frac{2}{27}}{x-1}+\frac{\frac{1}{9}}{(x-1)^2}+\frac{\frac{2}{27}}{x+2}+\frac{\frac{1}{9}}{(x+2)^2}.\tag{19.58} \end{equation}
which we can integrate.
