It is time to fill in that blank so we will try to “run the Power Rule backward,” if we can. In differential form Product Rule formula is
\begin{equation*} \dx(uv)=v\dx{u}+u\dx{v}\text{.} \end{equation*}
To run this backwards we integrate both sides giving
\begin{equation*} \int \dx(uv)=\int v\dx{u}+\int u\dx{v}\text{,} \end{equation*}
which is equivalent to
\begin{equation} \int u\dx{v}=\int \dx(uv)-\int v\dx{u}.\tag{19.71} \end{equation}
Observing that \(\int \dx(uv)=uv\text{,}\) yields the Integration by Parts formula (in differential form):
\begin{equation*} \int{}u\dx{v} =uv-\int{}v\dx{u}. \end{equation*}
