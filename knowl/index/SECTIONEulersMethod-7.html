Since we specified that the curve must pass through the point \((0, 1)\) we see that we need to find a function \(y(x)\) that satisfies the following two conditions:
\begin{align} \dfdx{y}{x} = y, \amp{}\amp{}\text{and}\amp{}\amp{} y(0)=1.\tag{7.6} \end{align}
A problem like the one stated in Formula (7.6) is called an Initial Value Problem (Ivp). We will give a formal definition below (Definition 7.3.0.1).
