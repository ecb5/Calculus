Definition 12.1.1.4. Horizontal Asymptotes.

  1. If \(\limit{x}{\infty}{f(x)}=A\) then the graph of the function \(f(x)\) has the horizontal asymptote, \(y=A.\)
    🔗
    🔗
  2. If \(\limit{x}{-\infty}{f(x)}=B\) then the graph of the function \(f(x)\) has the horizontal asymptote, \(y=B.\)
