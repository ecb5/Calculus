Everything we’ve just shown assumes that if \(p(x)\) and \(q(x)\) are known then \(r(x)\) and \(s(x)\) can be found so that \(r(x)\cdot p(x)+ s(x)\cdot q(x)=1\text{.}\) For arbitrary functions this is a very restrictive condition. However as we observed above, if \(p(x)\) and \(q(x)\) are polynomials having only trivial common factors then the existence of \(r(x)\) and \(s(x)\) is guaranteed. Thus for polynomials the Partial Fraction Decomposition is also guaranteed.
