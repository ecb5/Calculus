Example A.13.2.4. Heaviside’s Cover-up Method.

Suppose we wish to find the partial fractions decomposition of \(\frac{3x+1}{(x-1)(x-2)(x-5)}\text{.}\) As before, we have
\begin{equation} \frac{3x+1}{(x-1)(x-2)(x-5)}=\ \frac{A}{x-1}+\frac{B}{x-2}+\frac{C}{x-5} \text{.}\tag{A.18} \end{equation}
This leads to
\begin{equation*} 3x+1=A\left(x-2\right)\left(x-5\right)+B\left(x-1\right)\left(x-5\right)+C(x-1)(x-2) \end{equation*}
🔗
Observe that (A.18) will be true for every real (or complex) value of \(x\text{.}\) The factor \(x-1\) appears in two of the terms on the right suggests that we let \(x=1.\) In that case we have
\begin{equation*} 3\left(1\right)+1=A\left(1-2\right)\left(1-5\right) \end{equation*}
or
\begin{equation*} \frac{3\left(1\right)+1}{\left(1-2\right)\left(1-5\right)}=A. \end{equation*}
🔗

Aside: Comment.

Actually we are replacing \(x-1\) with 1.
🔗
We did not simplify this last formula in order to highlight the following\(.\) Observe that we get the same result if we simply “cover up” the factor \((x-1)\) on the left side of (A.18) above and set \(x=1.\)
\begin{equation*} A=\frac{3\left(1\right)+1}{(\text{covered})\left(1-2\right)\left(1-5\right)}=1. \end{equation*}
🔗
In precisely the same fashion (setting \(x=2\) and \(x=3\)) we get:
\begin{equation*} B=\frac{\left(3\right)2+1}{\left(2-1\right)\left(\text{covered}\right)\left(2-5\right)}=\ \frac{-7}{3}, \end{equation*}
and
\begin{equation*} C=\frac{\left(3\right)5+1}{(5-1)(5-2)(\text{covered})}=\frac{4}{3}, \end{equation*}
so that
\begin{equation*} \frac{3x+1}{(x-1)(x-2)(x-5)}=\frac{1}{x-1}+\frac{-7/3}{x-2}+\frac{4/3}{x-5}.\ \end{equation*}
🔗

Aside: Warning!

In order to use Heaviside’s Cover–Up Method you must have distinct linear factors in the denominator.
🔗
In this example they were \(x-1\text{,}\) \(x-2\text{,}\) and \(x-5\text{.}\) If our rational function had been
\begin{equation*} \frac{3x+1}{(x-1)^2(x-2)(x-5)} \end{equation*}
Heaviside’s Method would not work because \((x-1)^2\) is not linear.
🔗
Try it and see what goes wrong.
🔗
This trick is known as Heaviside’s Cover-Up Method (HCUM). It is well known, and can be used to find the PFD of a rational function as long as the denominator can be factored into distinct linear factors.
